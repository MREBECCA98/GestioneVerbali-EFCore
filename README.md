Gestione Contravvenzioni PM - Web Application

Questo progetto consiste nello sviluppo di una web application ASP.NET Core MVC per la gestione delle contravvenzioni della Polizia Municipale, utilizzando un database già esistente creato in precedenza.
L'applicazione consente di inserire e visualizzare dati sui trasgressori, le violazioni e i verbali, e di generare report aggregati sullecontravvenzioni.

Funzionalità principali:
-Gestioni trasgressori, possibilità di anagrafare i cittadini con dati: nome, cognome, indirizzo, città, CAP, codice fiscale
-Gestione violazioni, visualizzazione e gestione dei tipi di violazione contestabili
-Gestione verbali. inserimento dei verbali con data violazione, indirizzo, agente, importo e punti decurtati
-Report, dov'è possibile visualizzare i verbali trascritti raggruppati per trasgressore, il totale dei punti decurtati per trasgressore, la visualizzazione delle violazioni con più di 10 punti decurtati e la visualizzazione delle violazioni con importo maggiore di 400 euro

Tecnologie utilizzate:
-ASP.NET Core MVC
-SQL Server con Entity Framework Core
-HTML, Razor View, CSS
